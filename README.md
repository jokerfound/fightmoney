# FightMoney - 搜打撤类型网页游戏

一个基于 Phaser.js + TypeScript 开发的网页版2D"搜打撤"类型游戏。

## 🎮 游戏简介

FightMoney 是一款紧张刺激的生存掠夺游戏，核心玩法围绕**物资搜刮-战斗对抗-安全撤离**的循环。玩家需要在危险环境中搜集有价值的物资，与敌人交战，并成功撤离以保留战利品。

## ✨ 核心特性

- 🎯 **完整的游戏循环**：搜刮 → 战斗 → 撤离
- 💰 **经济系统**：物资收集、金币交易、仓库管理
- 🔫 **武器系统**：多种武器类型，不同伤害和特性
- 🛡️ **护甲系统**：提供伤害减免和生存能力
- 🎒 **背包系统**：物品管理、重量限制
- 🗺️ **地图生成**：随机房间布局，探索乐趣
- 👾 **敌人AI**：智能敌人，多种类型
- 🚪 **撤离系统**：撤离点开关机制，激活后开启撤离通道

## 🚀 快速开始

### 环境要求

- Node.js 16+ 
- npm 或 yarn

### 安装步骤

1. 克隆仓库
```bash
git clone <your-repo-url>
cd fightmoney
```

2. 安装依赖
```bash
cd phaser-game
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 在浏览器中打开 `http://localhost:5173`

### 构建生产版本

```bash
npm run build
```

构建后的文件在 `dist/` 目录中。

## 🎮 游戏操作

- **WASD** - 移动角色
- **鼠标左键** - 射击/攻击
- **R键** - 换弹
- **E键** - 拾取物品/交互
- **Tab键** - 打开/关闭背包
- **空格键** - 开始游戏

### 撤离流程

1. 找到中央大厅的**撤离开关**（红色开关）
2. 靠近开关，按 **E键** 激活
3. 开关激活后，撤离点房间会开启门口
4. 前往撤离点房间完成撤离
