# 搜打撤游戏 - 本地启动指南

## 快速启动

### Windows 用户
直接双击运行 `start.bat` 文件，脚本会自动：
1. 检查并安装依赖（如果未安装）
2. 启动开发服务器
3. 自动在浏览器中打开游戏

### Linux/Mac 用户
在终端中运行：
```bash
chmod +x start.sh
./start.sh
```

### 手动启动

1. **安装依赖**（首次运行需要）
```bash
npm install
```

2. **启动开发服务器**
```bash
npm run dev
```

3. **访问游戏**
- 游戏会自动在浏览器中打开
- 如果没有自动打开，请手动访问：http://localhost:3000

## 项目信息

- **开发端口**: 3000
- **技术栈**: Phaser.js + TypeScript + Vite
- **游戏类型**: 搜打撤类型生存掠夺游戏

## 功能说明

### 已实现功能
- ✅ 完整的物资系统（重量、稀有度、耐久度）
- ✅ 战斗系统（武器、弹药、护甲）
- ✅ 敌人AI（巡逻、追击、攻击）
- ✅ 撤离系统（倒计时、随机事件）
- ✅ 仓库管理系统（容量限制、升级）

### 游戏操作
- **移动**: WASD 或 方向键
- **射击**: 鼠标左键
- **换弹**: R键
- **切换武器**: 数字键1-4
- **拾取物品**: 靠近物品自动拾取
- **撤离**: 到达撤离点后按E键

## 常见问题

### 端口被占用
如果3000端口被占用，可以修改 `vite.config.ts` 中的端口号：
```typescript
server: {
  port: 3001, // 改为其他端口
}
```

### 依赖安装失败
- 检查网络连接
- 尝试使用国内镜像：`npm install --registry=https://registry.npmmirror.com`
- 清除缓存后重试：`npm cache clean --force`

### 游戏无法加载
- 检查浏览器控制台是否有错误信息
- 确保所有依赖都已正确安装
- 尝试清除浏览器缓存

## 开发命令

- `npm run dev` - 启动开发服务器
- `npm run build` - 构建生产版本
- `npm run preview` - 预览生产版本

## 注意事项

- 游戏数据保存在浏览器的 localStorage 中
- 清除浏览器缓存会删除游戏进度
- 建议使用 Chrome 或 Firefox 浏览器以获得最佳体验
